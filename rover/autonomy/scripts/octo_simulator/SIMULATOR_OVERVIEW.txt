🐛 TROUBLESHOOTING
───────────────────────────────────────────────────────────────────────────

Problem: "Module not found"
Solution: pip install numpy matplotlib PyQt5

Problem: Blank window
Solution: Update graphics drivers, try rotating 3D view

Problem: "No path found" unexpectedly
Solution: Enable occupancy grid, check console, verify positions

Problem: Slow performance
Solution: Reduce obstacles, disable grid view, lower speed

Problem: Path goes through obstacles
Solution: Check obstacle heights, verify point cloud generation

See SIMULATOR_README.md for complete troubleshooting guide.
